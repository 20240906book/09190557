做法： 1. 地瓜去皮切厚片蒸熟，加入糖趁熱壓成泥狀，再加入水、木薯粉，拌揉成耳垂軟度的粉糰。 2. 粉糰搓成長條，再分切成小塊，揉成小圓球。 3. 油溫不要太高（150度），入炸成半透狀，浮起後再按壓脹大，反覆按壓脹大（約12分鐘）。 即可撈出。2023年3月29日
